__Cara Setup__
```
apt update -y
```
```
apt install wget -y
```
```
wget -O setup "https://raw.githubusercontent.com/senowahyu62/Tool-Ip_forwarding_Ubuntu-Debian/main/setup"
```
```
bash setup
```

```
ipfrwd
```
